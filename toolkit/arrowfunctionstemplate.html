Arrow Function Challenge

This function is from my 157A final project. I used it to create a slideshow of my images. In this particular function, it is switching from one foto to the next.

Named Function

            function nextCPhoto() {
                fImage++; 
                if (fImage > cultureImages.length - 1) {
                    fImage = 0;
                }
                Cslide.src = `images/${cultureImages[fImage]}`;
            }

Function Expression

            const nextCPhoto = function() {
                fImage++; 
                if (fImage > cultureImages.length - 1) {
                    fImage = 0;
                }
                Cslide.src = `images/${cultureImages[fImage]}`;
            }

Arrow Function

            const nextCPhoto = () => {
                fImage++; 
                if (fImage > cultureImages.length - 1) {
                    fImage = 0;
                }
                Cslide.src = `images/${cultureImages[fImage]}`;
            }
